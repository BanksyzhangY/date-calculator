# 📅 在线日期工具集

在线日期计算器和工作日计算器，自动识别 2025–2026 年中国法定假期和调休安排。

## 工具链接

- **[日期计算器](https://banksyzhangy.github.io/date-calculator/date-calculator.html)** — 输入开始日期和工作日天数，推算目标日期
- **[工作日计算器](https://banksyzhangy.github.io/date-calculator/workday-calculator.html)** — 输入起止日期，计算区间内的工作日天数

## 数据来源

国务院办公厅放假通知。2026 年假期为预排安排，以官方正式通知为准。
